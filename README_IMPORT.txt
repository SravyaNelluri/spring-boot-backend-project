HOW TO IMPORT IN STS4
1. Extract this zip file.
2. Open Spring Tool Suite 4.
3. File -> Import -> Maven -> Existing Maven Projects.
4. Select the extracted coursemanagement folder.
5. Click Finish.
6. Open src/main/resources/application.properties and change MySQL username/password if needed.
7. In MySQL run: create database course_management;
8. Right click project -> Maven -> Update Project.
9. Right click project -> Run As -> Spring Boot App.
